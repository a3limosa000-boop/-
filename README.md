# Flask To-Do List App

A minimal, single-file Flask web application for managing a personal to-do list. All tasks are stored in memory while the server is running.

## What the Project Does

This app lets you keep track of tasks through a simple web page:
- Add new tasks with a title and a priority level (Low, Medium, High)
- Mark tasks as complete or undo that status
- Delete tasks you no longer need
- Tasks are automatically sorted so incomplete, high-priority items appear first
